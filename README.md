# Base
Estructura base para proyectos PHP

## Instalación
Crear base de datos y agregar credenciales al archivo '/vendor/nisadelgado/framework/database.php'
~~~
git clone https://github.com/nisa6delgado/base.git
cd base
composer install
npm install
php -S localhost:8000
~~~
